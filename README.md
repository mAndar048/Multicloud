# CloudPilot

CloudPilot is a multi-cloud infrastructure automation tool that converts plain-English deployment intent into deterministic Terraform-driven deployments.

## Stack
- Python 3.11+
- FastAPI
- Terraform
- Celery + Redis
- YAML knowledge base
- Streamlit

## Current Status
This repository is scaffolded for phased implementation based on [CLOUDPILOT_IMPLEMENTATION_PLAN.md](CLOUDPILOT_IMPLEMENTATION_PLAN.md).

## Quick Start
1. Create and activate a virtual environment.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Copy environment variables:
   ```bash
   cp .env.example .env
   ```

## Run Services
```bash
uvicorn cloudpilot.main:app --reload
```

```bash
celery -A cloudpilot.jobs.worker worker --loglevel=info
```

```bash
streamlit run cloudpilot/ui/app.py
```

## Run Tests
```bash
pytest cloudpilot/tests -q
```

## Implementation Notes
- `IntentObject` in `cloudpilot/intent/schema.py` is the shared contract across modules.
- Terraform templates are selected and parameterized; they are not generated by LLM output.
- The conversation layer is implemented as an explicit state machine.
